如何使用angular

1.构建一个html页面，引入angularjs类库(1.5的版本)，下载地址： AngularJS by Google


2.ng-app定义angularjs的控制范围，如：data-ng-app="MetronicApp"


3.ng-controller定义angularjs的控制范围，如：ng-controller="metronicCtrl"


4.在js中定义模块(module)和控制器(controller) 

				var metronicApp = angular.module("MetronicApp", []); 	//定义模块，注意与angular.module("MetronicApp")的区别
				
				//注意区别：metronicApp.controller('metronicCtrl', function($scope, $rootScope) { ... });
				metronicApp.controller('metronicCtrl', ['$scope', '$rootScope', function($scope, $rootScope) {
					//do something ...
				}]);


5.完了。但加点解释：
angular是一个内置对象，可以通过angular.module("MetronicApp", [])方法来构建一个模块。
在模块方法metronicApp.controller('metronicCtrl',function( ... ){ ... })构造一个controller，负责控制ng-controller='metronicCtrl'的视图范围
