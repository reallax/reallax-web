服务

什么是服务？

1.服务是公用功能代码的集合，也是粘结多个controller的胶水
2.Lazily instantiated – Angular only instantiates a service when an application component depends on it.
3.Singletons – Each component dependent on a service gets a reference to the single instance generated by the service factory.


如何定义一个服务？

定义一个服务的方式有三种：service，factory，provider。
angular.module('moduleName').service('serviceName', function( injectables ) { ... }); --可以通过serviceName注入
angular.module('moduleName').factory('factoryName', function( injectables ) { ... }); --可以通过factoryName注入
angular.module('moduleName').provider('providerName', function( injectables ) { ... });--可以通过providerName注入
service，factory创建服务，最终都是调用的provider；换句话说，前两者是provider的简易使用方式，能让开发者快速上手。区别及说明，请看示例。
